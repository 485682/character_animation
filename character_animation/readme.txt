
* compiled with Microsoft DirectX SDK June 2010
* installed at [  C:\Program Files\Microsoft DirectX SDK (June 2010)  ]


* to generate the a _mesh file, use the collada_loader application
* collada_loader usage .. [ app ] [ file.dae ] [ file._mesh ]
* colladaloader was written for blender 2.76 

*       to generate a .dae file 
* export only one mesh with one armature  
* mesh must have texcoords, Normals and of course vertices.
* more than 3 vertex attributes (for example vertex colors) will  generate an error

* to toggle vsync         press v key
* to toggle fullscreen    press f key
* to switch animations    press n key
